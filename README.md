# Food Express Delivery Platform

A Java Swing and MySQL database application for managing a multi-vendor food delivery platform.

- For setup and execution steps see: docs/run_instructions.txt.
- For USER GUIDE, SQL EVIDENCE, & PRESENTATION SLIDES see docs folder.

Technologies Used:

- Java Swing
- JDBC
- MySQL Workbench

Features:

- Customer ordering
- Restaurant order management
- Driver delivery tracking
- Admin management dashboard
